Bases de Dados

Uma Base de Dados é uma colecção de dados partilhados, inter-relacionados e usados par amúltiplos objectivos.

Dados “Factos conhecidos” que podem ser armazenados e que possuem um significado implícito.


Propriedades de uma BD

    -Uma BD é uma colecção lógica coerente de dados com um significado inerente.

    -Disposição desordenada dos dados não pode ser referenciada como uma bd.

    -É projectado, construído para um propósito específico. Ele possui um conjunto pré definido de utilizadores e aplicações.

    -BD representa algum aspecto do mundo real – “Mini Mundo”
